# Innomatics Hackathon

This project was done as part of the Innomatics data analytics hackathon.

## Project Overview
The goal of this project is to analyze food delivery data by combining multiple datasets and extracting useful insights.

## Datasets Used
- orders.csv (order transactions)
- users.json (user details)
- restaurants.sql (restaurant details)

## Tools & Technologies
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Work Done
- Merged CSV, JSON, and SQL datasets into a single dataset
- Performed data analysis on revenue, orders, cities, cuisines, and memberships
- Answered MCQs and fill-in-the-blank questions using data insights

## Learning Outcome
- Gained hands-on experience with real-world datasets
- Improved skills in data cleaning, merging, and analysis
